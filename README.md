# Epub Maker

**献给所有喜爱网文的人**

`Epub Maker` 是一个基于 Python `tkinter` 的桌面工具，用来把整个 TXT 文件按标题层级拆分，并导出 HTML 或直接生成 EPUB。

## 功能简介

- 读取 TXT 文件，自动尝试常见中文编码
- 在界面中直接查看和编辑全文
- 按 `第X<关键字>` 规则识别多级标题
- 支持通过 `H1-H4` 自定义层级关键字，例如：`卷 / 章 / 节 / 部`
- 左侧列表显示编号、级别、标题和预警
- 点击左侧标题可在正文中快速定位
- 支持正文关键字搜索和上一个 / 下一个跳转
- 使用模板导出 HTML 文件
- 支持填写书名、作者、分类
- 支持选择封面图片并直接生成 EPUB 3

## 当前标题识别规则

程序不会模糊匹配标题，而是按固定规则识别：

- 标题格式必须符合 `第X<关键字>`
- `X` 支持阿拉伯数字和常见中文数字
- `<关键字>` 由右侧 `H1-H4` 输入框决定
- 留空的层级不会参与识别

例如：

- `第一卷`
- `第12章 初见`
- `第三节`

## 模板逻辑

程序目前有两套模板来源：

### 1. HTML 内容模板

程序使用 `template` 目录下的两个模板：

- `group.html`
- `content.html`

导出时的规则是：

- `H1` 标题如果下面没有正文，使用 `group.html`
- 其他情况使用 `content.html`
- `content.html` 中的标题标签会根据层级自动映射为 `h1-h4`

程序初次运行时，如无 `template` 目录，则自动生成目录及默认模板。可根据需要自行修改。

### 2. EPUB 结构模板

程序会读取根目录下的 `模板.epub` 作为 EPUB 骨架。

生成 EPUB 时，程序会：

- 解包 `模板.epub`
- 把章节内容写入 `OEBPS/Text/`
- 更新 `OEBPS/Text/nav.xhtml`
- 更新 `OEBPS/Text/coverpage.xhtml`
- 更新 `OEBPS/content.opf`
- 把用户选择的封面图复制到 `OEBPS/Images/cover.xxx`
- 最后重新打包为新的 `.epub`

因此：

- 想改 EPUB 的 CSS，请直接修改 `模板.epub` 内的样式文件
- 想增加分隔线等静态图片，请直接放进 `模板.epub` 的 `OEBPS/Images/`
- 只要模板里的 XHTML 或 CSS 已经引用这些资源，生成时就会一起保留

## 文本处理

当前程序只做比较轻量的文本处理：

- 尝试多种编码读取 TXT
- 统一换行符为 `\n`
- 分析时去掉每行首尾空白
- 跳过空行
- 导出 HTML 时进行 HTML 转义

不会自动做这些事：

- 不会自动去广告
- 不会自动合并断行
- 不会自动删除网站尾注

## 预警规则

程序会对部分疑似异常标题给出预警，主要包括：

- 标题编号不连续
- 标题中包含常见杂项词，例如求票、感谢、PS 等
- 标题下一行像是残句或异常标点开头

状态栏会显示：

- 标题总数
- 预警数
- 各级标题数

## 使用方法

1. 点击“读取TXT”导入小说文本
2. 在右侧设置 `H1-H4` 层级关键字
3. 点击“分析章节”
4. 在左侧查看识别结果，必要时直接修改中间正文
5. 如需生成 EPUB，填写 `标题 / 作者 / 分类`
6. 如需生成 EPUB，点击“选择封面图片”
7. 点击“生成HTML”或“生成EPUB”

## 输出目录

- HTML 模板目录：`template/`
- HTML 导出目录：`html/`
- EPUB 导出目录：`epub/`
- EPUB 模板文件：根目录下的 `模板.epub`

生成后的文件名按顺序编号，例如：

- `0001.html`
- `0002.html`
- `0003.html`

生成 EPUB 时，章节文件会写为：

- `0001.xhtml`
- `0002.xhtml`
- `0003.xhtml`

分类支持多个值，可使用中文逗号 `，` 或英文逗号 `,` 分隔。

## 打包后的使用说明

如果打包为 exe，建议把以下文件放在 exe 同目录：

- `模板.epub`

`template/` 目录不是必须手动提供：

- 如果存在，程序优先使用你自定义的 `group.html` / `content.html`
- 如果不存在，程序会在首次运行时自动生成默认模板

## 运行环境

- Python 3
- 标准库依赖：`tkinter`、`os`、`re`、`html` 等

## exe编译方法
该命令适用于 Windows 环境
```
pyinstaller --add-data "epub maker.ico;." -F -w -i "epub maker.ico" "epub maker.py"
```

编译完成后，如需使用“生成 EPUB”功能，请确保 `模板.epub` 与 exe 位于同一目录。
