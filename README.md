# Epub Maker

**献给所有喜爱网文的人**

`Epub Maker` 是一个基于 Python `tkinter` 的桌面工具，用来把整个 TXT 文件按标题层级拆分，并导出为一组 HTML 文件，方便后续制作 EPUB。

## 功能简介

- 读取 TXT 文件，自动尝试常见中文编码
- 在界面中直接查看和编辑全文
- 按 `第X<关键字>` 规则识别多级标题
- 支持通过 `H1-H4` 自定义层级关键字，例如：`卷 / 章 / 节 / 部`
- 左侧列表显示编号、级别、标题和预警
- 点击左侧标题可在正文中快速定位
- 支持正文关键字搜索和上一个 / 下一个跳转
- 使用模板导出 HTML 文件

## 当前标题识别规则

程序不会模糊匹配标题，而是按固定规则识别：

- 标题格式必须符合 `第X<关键字>`
- `X` 支持阿拉伯数字和常见中文数字
- `<关键字>` 由右侧 `H1-H4` 输入框决定
- 留空的层级不会参与识别

例如：

- `第一卷`
- `第12章 初见`
- `第三节`

## 模板逻辑

程序使用 `template` 目录下的两个模板：

- `group.html`
- `content.html`

导出时的规则是：

- `H1` 标题如果下面没有正文，使用 `group.html`
- 其他情况使用 `content.html`
- `content.html` 中的标题标签会根据层级自动映射为 `h1-h4`

程序初次运行时，如无 `template` 目录，则自动生成目录及模板。可根据需要自行修改模板。

## 文本处理

当前程序只做比较轻量的文本处理：

- 尝试多种编码读取 TXT
- 统一换行符为 `\n`
- 分析时去掉每行首尾空白
- 跳过空行
- 导出 HTML 时进行 HTML 转义

不会自动做这些事：

- 不会自动去广告
- 不会自动合并断行
- 不会自动删除网站尾注

## 预警规则

程序会对部分疑似异常标题给出预警，主要包括：

- 标题编号不连续
- 标题中包含常见杂项词，例如求票、感谢、PS 等
- 标题下一行像是残句或异常标点开头

状态栏会显示：

- 标题总数
- 预警数
- 各级标题数

## 使用方法

1. 点击“读取TXT”导入小说文本
2. 在右侧设置 `H1-H4` 层级关键字
3. 点击“分析章节”
4. 在左侧查看识别结果，必要时直接修改中间正文
5. 点击“生成HTML”

## 输出目录

- 模板目录：`template/`
- 导出目录：`html/`

生成后的文件名按顺序编号，例如：

- `0001.html`
- `0002.html`
- `0003.html`

## 运行环境

- Python 3
- 标准库依赖：`tkinter`、`os`、`re`、`html` 等

## exe编译方法
```
pyinstaller --add-data "epub maker.ico;." -F -w -i "epub maker.ico" "epub maker.py"
```
